# Página de Inventário para o Cadastro de Clientes e Produtos

Plataforma de inventário para o cadastro de clientes e produtos - Processo do Hiring Coders

## Tecnologiais

* HTML 5
* CSS / SASS
* Bootstrap
* JQuery
* Single Page Application

**Obs:** A aplicação garante que em inputs específicos o usuário poderá colocar apenas números e que no campo de "Email" o input precisa ser do formato de um email.
